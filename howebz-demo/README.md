# Howebz Homepage — Final Subscription Version

This version changes the homepage positioning from an AI tool to a done-for-you website and monthly marketing service.

## Files

- `index.html` — page structure and final sales copy
- `styles.css` — responsive desktop and mobile design
- `script.js` — navigation, accordions, caption expansion, plan comparison and tooltips
- `howebz-homepage-single.html` — all HTML, CSS and JavaScript combined into one file
- `assets/` — continue using your existing assets folder

## Main sections

1. Hero
2. How It Works
3. What We Prepare For You
4. Subscription Plans
5. All Plans Include
6. Full Plan Comparison
7. Monthly Content Rules
8. Monthly Service Process
9. FAQ
10. WhatsApp CTA

## Open locally

Keep this folder next to your existing `assets` folder, then open `index.html`.

For a local server:

```bash
python -m http.server 8080
```

Open:

```text
http://localhost:8080
```
